# v0.16.0

* Upgrade to Chrome 37.
* Add support for Chrome devtools extension.

* Fix `BrowserWindow.setResizable`.
* Fix `BrowserWindow.setSkipTaskbar`.
* Set `WM_CLASS` and `WM_WINDOW_ROLE` for browser window.
